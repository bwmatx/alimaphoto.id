PRD SECTION 2 — RECENT PROJECT CAROUSEL
ALIMA PHOTO

1. Tujuan Section
   Section ini berfungsi sebagai showcase portofolio utama yang menampilkan hasil karya terbaru Alima Photo dalam format visual yang kuat, premium, dan langsung menarik perhatian. Fokus utamanya adalah memperlihatkan kualitas proyek, bukan banyak teks. Section ini harus terasa seperti galeri editorial yang elegan, modern, dan profesional.
   Section ini menjadi jembatan visual antara hero/home section dan section berikutnya, sehingga alur scrolling tetap terasa hidup dan berkelas.
2. Arah Desain
   Carousel harus mengikuti gaya visual seperti contoh gambar:
   • Susunan kartu vertikal / portrait
   • Scroll utama ke kanan, bukan ke atas
   • Tampilan full width atau hampir full width
   • Setiap kartu memuat satu project
   • Kartu dibuat tinggi dan ramping seperti panel galeri
   • Gaya keseluruhan terasa cinematic, luxury, dan editorial
   Tidak perlu menampilkan heading atau title section seperti “Recent Project” karena visual kartu sudah cukup kuat sebagai fokus utama.
3. Struktur Carousel
   Carousel menggunakan format horizontal scrolling strip dengan perilaku berikut:
   • Bergerak ke kanan secara otomatis
   • Bisa di-drag atau swipe secara manual
   • Bisa looping tanpa putus
   • Transisi antar kartu harus halus
   • Pergerakan terasa natural dan premium
   • Tidak menggunakan efek yang terlalu cepat atau agresif
   Carousel harus memberi kesan seperti deretan karya yang sedang dipamerkan di galeri fotografi kelas atas.
4. Jumlah Konten Carousel
   Total item carousel:
   • 7 project card
   • 5 project utama
   • 2 project tambahan untuk mendukung looping seamless
   Konten utama yang ditampilkan harus tetap konsisten dan representatif untuk portofolio Alima Photo.
5. Informasi pada Setiap Kartu
   Setiap project card wajib menampilkan:
   • Nama project / client
   • Jumlah foto di project tersebut
   Format penempatan teks:
   • Nama project/client diletakkan di bagian bawah gambar
   • Jumlah foto ditampilkan tepat di bawah nama project/client
   • Teks dibuat jelas, ringkas, dan mudah dibaca
   Contoh struktur isi kartu:
   • Andini & Rizky Wedding
   o 128 Photos
   • Prewedding Dinda & Reza
   o 74 Photos
   • Engagement Project
   o 52 Photos
   Tampilan teks harus minimalis, elegan, dan menyatu dengan foto.
6. Visual Behavior
   Section ini harus memiliki karakter visual sebagai berikut:
   • Portrait cards tinggi
   • Foto dominan, teks sekunder
   • Kontras tinggi agar informasi tetap terbaca
   • Hover state lembut
   • Fokus pada storytelling visual
   • Tidak crowded
   • Tidak memakai title besar di atas carousel
   Seluruh kartu harus konsisten dalam aspek visual agar terlihat seperti satu rangkaian proyek yang kurasi dengan baik.
7. Motion & Interaction
   Animasi carousel harus terasa:
   • Halus
   • Mewah
   • Stabil
   • Tidak patah-patah
   • Tidak terlalu cepat
   Sistem animasi yang diinginkan:
   • Auto-scroll horizontal ke kanan
   • Infinite looping
   • Smooth transition antar item
   • Optional pause ketika hover
   • Optional drag interaction untuk desktop
   • Swipe friendly untuk mobile
   Kalau user berhenti di sebuah kartu, konten tetap mudah dibaca tanpa gangguan visual berlebihan.
8. Content Strategy
Karena section ini adalah portofolio utama, isi project harus merepresentasikan kualitas terbaik Alima Photo. Fokus konten:
• Wedding
• Prewedding
• Engagement
• Studio
• Maternity

9. UX Goal
   User harus merasakan bahwa:
   • Alima Photo punya karya yang nyata dan berkualitas
   • Setiap project punya identitas tersendiri
   • Portofolio disajikan secara profesional
   • Website terasa premium dan modern
   • Section ini mendorong user untuk lanjut melihat layanan atau langsung booking
10. Technical Notes
    Untuk implementasi section ini:
    • Gunakan layout horizontal scroll
    • Gunakan elemen portrait card
    • Pastikan responsif di desktop, tablet, dan mobile
    • Gunakan lazy loading untuk gambar
    • Pastikan teks tetap terbaca di semua ukuran layar
    • Jaga performa agar carousel tetap ringan
